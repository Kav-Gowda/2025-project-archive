# Health Prognosis AI

## Overview

This project implements machine learning and deep learning models to predict trends in COVID-19 case data.

It demonstrates time series forecasting using both traditional models and neural networks.

---

## What I Implemented

* Data preprocessing for real-world healthcare datasets
* Time series analysis with correlation and lag features
* Linear regression baseline model
* Neural network and LSTM-based forecasting models
* Model evaluation and visualization

---

## Tech Stack

* **Libraries:** Pandas, NumPy, scikit-learn
* **Deep Learning:** TensorFlow / Keras
* **Visualization:** Matplotlib, Seaborn
* **Language:** Python

---

## How It Works

1. Load and preprocess COVID-19 time series data
2. Analyze trends and correlations
3. Train baseline (Linear Regression) model
4. Train neural network and LSTM models
5. Compare predictions and visualize results

---

## Key Features

* Time series forecasting with multiple models
* Comparison of traditional ML vs deep learning
* Use of LSTM for sequential data modeling
* Visualization of predictions and model performance
